# Face Filter App

This app uses a face filter library to apply 3D models (like hats and glasses) to a user's face in real-time. It leverages React Three Fiber for 3D rendering and the Jeeliz Face Filter library for face tracking.

## Features

- Real-time face tracking
- 3D model application (hats, glasses, etc.)
- Responsive design

## Getting Started

1. Clone the repository
2. Install dependencies
3. Start the development server

```bash
git clone <repository-url>
cd <repository-directory>
npm install
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

- Select a 3D model from the available options
- Allow camera access when prompted
- Enjoy the face filter effects!
